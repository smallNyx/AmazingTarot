## AI塔罗牌网站开发计划

### 技术栈

* **前端**: React + TypeScript

* **后端**: Python + LangChain + FastAPI

* **AI**: DeepSeek API

* **数据库**: SQLite

* **认证**: JWT

* **部署**: Docker + DockerHub + GitHub Actions

### 塔罗牌阵详情

#### 基础牌阵（Phase 4）

**三张牌占卜法**
- 牌数: 3张
- 适用场景: 综合分析、单事解析
- 特点: 通用型占卜牌阵，可以自由定义应用在很多场合，不受约束的占卜相关事物。或用来分析独立事情的某个方面。本牌阵是万能型牌，可以应用在所有场合，常用于简单事物的占卜，根据牌面进行解析，直指核心。普通占卜时应用本牌阵，可以起到很好的效果，初学者适用。

#### 时间系列牌阵（Phase 5）

**时间流牌阵**
- 牌数: 3张
- 位置: 过去/现在/未来
- 适用场景: 预测未来、窥探未知
- 特点: 平行流向的时间解析法，纯粹的时间流向。应用于空间维度上的时间占卜。恍若流淌的时间从过去延伸到未来，事件平铺于时间毯之上。本牌阵适合有时间指向的占卜，是解析未来的利器，不适宜深层次占卜。你可以在时间轨道上占卜许多事情，任何事都有时间属性。

**圣三角牌阵**
- 牌数: 3张
- 位置: 过去/现在/未来 + 建议
- 适用场景: 判断情势、寻找成因
- 特点: 梳理问题前因后果，是时间流的变形。更注重事物内在原因，而非单纯时间流向。尤其适合根据因由来判断事物。本牌阵适合理清事情的原委因由，常用于单项独立事物的占卜，前因后果简洁扼要，清晰明了。尤其在研究事情脉路，针对审证求因的问题，有着良好的果效。

#### 自我探索牌阵（Phase 5）

**身心灵牌阵**
- 牌数: 3张
- 位置: 身/心/灵
- 适用场景: 自我探索、了解自己
- 特点: 通过身心灵可以很好的了解自身状况，从灵性、心理、身体三方面透彻审视自我，进而更真切明了自身的存在。本牌阵适合在自己迷惘的时候应用，向内寻求答案，比向外更有效。我们从哪来，将去往何地，让塔罗告诉你答案。

**直指核心牌阵**
- 牌数: 3张
- 位置: 核心/原因/建议
- 适用场景: 问题探索、切中要害
- 特点: 占卜问题的核心因素，适用于具体问题在遇到瓶颈的时候寻求突破。可以清楚问题的根本的所在，进而做出正确决策。本牌阵对解决问题帮助很大，可快速找到问题的症结所在，有着极强的问题解决能力。

#### 元素与关系牌阵（Phase 5）

**四元素牌阵**
- 牌数: 4张
- 位置: 火/水/风/土
- 适用场景: 问题探索、多方解析
- 特点: 通过四元素了解问题多方面的状况，从感性、理性、物质、行动四方面透彻审视，多重角度出发，进而了解问题实质。本牌阵适合了解自己对世界的认知，从不同的角度审视，这包括固定的问题或者单一的事物。如果你不能全面的了解它，就无法解决它。

**恋人金字塔**
- 牌数: 5张
- 位置: 关系分析
- 适用场景: 占卜恋人关系、互动解析
- 特点: 简洁直接，涵盖两人相恋原始要素，适合恋人情侣间的占卜，牌面一出明了易懂，是占卜爱情方面常用的牌阵。本牌阵要素明确，条理清晰，适合恋人之间关系的占卜能够预测未来的发展趋势。两个人之间遇到矛盾亦或迷惘，可以通过这个牌阵占卜未来的走向。

#### 爱情系列牌阵（Phase 5）

**爱情大十字**
- 牌数: 5张
- 位置: 十字布局
- 适用场景: 解析两性关系、爱情状况
- 特点: 注重内心情感，主要应用于情侣之间，善于洞悉彼此关系中的情感状况并分析结果。本牌阵布局合理清晰，用于感情心理的占卜可以为感情生活中带来帮助。如果你想解决感情问题，不妨试一下这个牌阵。

**寻找对象牌阵**
- 牌数: 4张
- 位置: 自我认知/期望/挑战/建议
- 适用场景: 寻找意中人、有缘人
- 特点: 用于未来爱人的占卜，适宜寻找什么样的男朋友、女朋友。先让塔罗帮助做好准备，再出发。本牌阵适合单身人士占卜，可用来健全意中人的愿景，帮助自己确定目标。不知道自己想要什么，不妨问一问这个牌阵吧。

**爱情树牌阵**
- 牌数: 7张
- 位置: 树形结构
- 适用场景: 溯本求源、寻找症结
- 特点: 解析前因后果，回溯爱情过往的牌阵，探究本源核心，指引未来趋势，为陷入爱情泥沼的情侣提供改善建议。本牌阵适合恋爱遇到困境时做占卜，寻找潜在原因，改善感情关系。当你遇到感情困境，可用本牌阵突出重围。

**真命天子牌阵**
- 牌数: 5张
- 位置: 内在/外在分析
- 适用场景: 单身探索有缘人、合意对象
- 特点: 真命天子更多指的是情投意合的人，通过探索自己内心的想法，改善外在的行为，遇到那个适合的人。本牌阵适合单身人士邂逅有缘人的占卜，通过内在和外在的双重改观，找到合适的伴侣。

**吉普赛牌阵**
- 牌数: 5张
- 位置: 双方分析 + 关系
- 适用场景: 情侣分析、关系延展
- 特点: 恋爱中人专用牌阵，对双方各自剖析，对比相处方式和环境适应能力，总结恋爱中遇到的的问题，做出对未来的预判。本牌阵适合婚姻、爱情、恋爱、感情方面的占卜，探索彼此内心想法，找到合适的相处方式。

**爱之星牌阵**
- 牌数: 5张
- 位置: 爱情分析
- 适用场景: 占卜爱情、两性关系
- 特点: 爱情关系塔罗牌阵，专门用于占卜爱情，可以分析两个人的心理状况，对未来的期望，以及分析两性关系的最终结果。本牌阵适合遇到感情问题所进行的占卜，理清感情问题的前因后果，促进关系融洽。

**婚姻牌阵**
- 牌数: 6张
- 位置: 婚姻分析
- 适用场景: 结婚运势、婚姻剖析
- 特点: 针对婚姻状况和期望进行解析，通过牌面分析姻缘，把握婚姻走势，根据现实情况，推导出最后的结果。本牌阵适合婚姻专项的占卜，探明姻缘脉动，解析婚姻关系。尤其在婚前婚后，了解婚姻状况时适合应用本牌阵。

**恋人树牌阵**
- 牌数: 6张
- 位置: 树形分支
- 适用场景: 探究对方心理、行为模式
- 特点: 详细分析对方的心理和态度，可以找到存在于两人之间的症结利弊，细节揣摩，做出正确的选择。本牌阵通过树形分支，可对比情侣间的行为差异，进而根据事实做出判断。

**维纳斯牌阵**
- 牌数: 8张
- 位置: 未来指向布局
- 适用场景: 占卜爱情发展、预测走向
- 特点: 维纳斯牌阵剑指未来，有着超然的力量，预测爱情直指核心，真实快速的切入未来，是难得的经典塔罗牌阵。本牌阵适合婚姻、恋爱方面的未来指向占卜，是分析爱情未来的牌阵，可以看到双方未来的状况。

**情人复合牌阵**
- 牌数: 5张
- 位置: 复合分析
- 适用场景: 分手复合、前任想法
- 特点: 情人复合牌阵，针对分手复合的占卜，通过牌面分析自己的状况和前任的想法，理清彼此关系，综合分析判断，做出选择。本牌阵适合念念不忘、觉得彼此前缘未了的情况。

#### 事业与财富牌阵（Phase 5）

**财富之树**
- 牌数: 6张
- 位置: 财富分析
- 适用场景: 事业发展、财运状况
- 特点: 解析财富运势生长的牌阵，财富之树占卜法用来占卜事业和财运，通过对财富的梳理，健全适合自己的财富模式，是诠释财运的好牌阵。本牌阵象征财富的生成，可揭示财富脉搏，对求财有积极的指导意义。

#### 选择决策牌阵（Phase 5）

**二选一牌阵**
- 牌数: 2张
- 位置: 选项A/选项B
- 适用场景: 主要用于抉择、判断
- 特点: 塔罗二选一适用于二种情况选择其中一种，在判断情势、决定方向等解析中应用广泛。本牌阵适合在两种情况中选择其中一种，可用于感情、事业、学业方面的占卜，用途相对广泛。

**三选一牌阵**
- 牌数: 3张
- 位置: 选项A/选项B/选项C
- 适用场景: 事情抉择、选择占卜
- 特点: 针对三个不同的选项进行占卜，通过牌面对利害关系进行分析判断，权衡利弊，做出最终的选择。本牌阵适合在较为复杂的环境中，做出正确的判断，分清各方优劣。

**问题解决牌阵**
- 牌数: 5张
- 位置: 问题/原因/方案
- 适用场景: 问题剖析、答疑解惑
- 特点: 用于解决具体问题的牌阵，通过对问题的剖析，对比前因后果，根据逻辑关系，提高处理问题的能力。总结问题评判损益得失，做出对未来的预判。

#### 人际关系牌阵（Phase 5）

**灵感对应牌阵**
- 牌数: 4张
- 位置: 双向互动
- 适用场景: 占卜情感因应、人际关系
- 特点: 擅长占卜情绪联接或情感互动，简约直观，有着神奇的应验效果。对于情侣、同事、朋友间的应用较为广泛。本牌阵适合婚姻、爱情、恋爱、感情方面的互动关系占卜，是双向的占卜牌阵，可以看到同层次的交融。

#### 运势系列牌阵（Phase 5）

**周运势牌阵**
- 牌数: 7张
- 位置: 周一至周日
- 适用场景: 周运分析、单周占卜
- 特点: 针对一周七天进行分析，综观周运势，逐天进行分析，提前做好预判。也可以通过整体牌面来预测本周运势的高低。本牌阵是周运占卜的专用牌阵，适用于周运占卜，可以占卜下一周的运势。

**六芒星牌阵**
- 牌数: 6张
- 位置: 六芒星布局
- 适用场景: 占卜事物发展、预测未来
- 特点: 六芒星牌阵意义深远，是指向未来的牌阵，预判事物的发展方向，理清事情的本源，是真正拥有预测属性的塔罗牌阵。本牌阵判断事情走向，有积极的指导意义，可分析潜意识与显意识的表达，有着极强的窥视未来能力。

#### 经典综合牌阵（Phase 5）

**凯尔特十字牌阵**
- 牌数: 10张
- 位置: 十字扩展布局
- 适用场景: 事件整体分析、宏观审视
- 特点: 经久弥新的古老塔罗牌阵，永恒经典、备受推崇的凯尔特十字牌阵。通过对事物的层层剖析，抽丝剥茧，帮助做出抉择。本牌阵拥有很强的总结和规化能力，结构细致严谨，可通过宏观角度审视事件全貌，协助做出有利的决策。
